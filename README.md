## 🖥️ Projeto
Esse é um projeto Web Responsivo de duma cápsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias
Esse Projeto foi desenvolvido durante o NLW da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
